# 欢迎来到我的数字花园 🌱

你好！欢迎来到我的个人博客。这里是十方空烬（voidemper）的数字花园，我将在这里记录我的技术探索、项目经验和生活思考。

## 🚀 为什么创建这个博客？

主要有几个原因：

1. **记录与整理**：在学习过程中，我会遇到各种有趣的知识点，通过写作可以更好地消化和整理
2. **分享与交流**：希望我的经验能够帮助到同样在探索技术的人
3. **个人成长**：写作是最好的思考方式之一
4. **GitHub 小绿点**：是的，这确实是一个有趣的动力 😄

## 🛠️ 技术栈选择

这个博客基于以下技术构建：

- **Astro**：现代静态站点生成器，性能优秀
- **TypeScript**：类型安全，开发体验好
- **Tailwind CSS**：原子化CSS，快速构建美观界面
- **GitHub Pages**：免费托管，自动部署

整个项目代码开源在 [GitHub](https://github.com/widechaos/widechaos-blog)。

## 📝 我会写什么？

这里主要会记录：

### 技术相关
- 编程语言与框架（Python, TypeScript, FastAPI 等）
- 系统设计与架构
- 量化交易系统开发
- AI 与机器学习应用

### 项目经验
- 个人项目的开发历程
- 遇到的问题和解决方案
- 最佳实践总结

### 生活思考
- 学习方法的探索
- 效率工具的分享
- 个人成长的反思

## 🎯 未来计划

我计划为这个博客添加以下功能：

1. **评论系统**：集成基于 GitHub 的评论
2. **搜索功能**：全文搜索
3. **RSS 订阅**：方便读者订阅更新
4. **多主题切换**：更多的配色方案
5. **数据分析**：了解读者偏好
