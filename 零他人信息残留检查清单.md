title: 零他人信息残留检查清单
---

## 目标

- 代码、内容、静态资源中不包含任何与站点作者无关的姓名、头像、社交链接、联系方式或第三方个人履历信息

## 1. 全库关键词扫描

建议在项目根目录执行以下检查（命中则必须确认并清理）：

```bash
rg -n "Lautaro|Rodriguez|Collins|lautarodevelops|linkedin\\.com/in/|github\\.com/" .
rg -n "Personal Portfolio|About Me|Sobre Mí|Proyectos Destacados" .
```

## 2. 站点配置核对

- [ ] [site.ts](file:///c:/Users/Administrator/OneDrive/Projects/widechaos-blog/src/config/site.ts) 中 `title/description/author/location/email/phone` 均为本人信息
- [ ] `social` 中没有他人账号链接（为空或为本人账号）
- [ ] 导航菜单与页面路由一致（`/blog`、`/archive`、`/tags`、`/about`、`/proyectos`、`/contact`）

## 3. 页面与组件核对

- [ ] 页头/页脚不包含他人姓名、头像或外链
- [ ] About/Contact 页面文案无他人经历描述
- [ ] 复制邮箱按钮内容为本人邮箱

## 4. 内容核对

- [ ] `src/content/blog/*.md` 文章 `author` 字段为本人
- [ ] 文章正文不包含他人项目链接、他人社媒链接、或他人版权/署名信息
- [ ] `draft: true` 的文章不会出现在 RSS/Sitemap（如需要“下线”，优先用 draft）

## 5. 静态资源核对

- [ ] `public/` 下不包含他人头像/他人项目截图（尤其是含姓名水印的图片）
- [ ] favicon 等品牌资源为本站使用

## 6. 构建产物抽检

执行：

```bash
npm run build
```

抽检 `dist/`：

- [ ] `dist/rss.xml` 不包含他人信息
- [ ] `dist/sitemap.xml` URL 列表正确，且无异常外链
- [ ] 任意文章页 HTML 中不包含他人姓名或社交链接
